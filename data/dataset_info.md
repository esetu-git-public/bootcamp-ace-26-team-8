## Dataset Overview

This dataset contains historical loan application records collected from financial institutions. It is designed to build a machine learning model capable of predicting whether a borrower is likely to default on a loan.

The dataset is suitable for supervised machine learning and binary classification tasks.

---

## Number of Rows

**255,347**

---

## Number of Columns

**18**

---

## Target Column

**Default**

Target Values:

- **0** → Loan Repaid (Non-Default)
- **1** → Loan Default

---

## Features

The dataset contains borrower information, financial details, and loan-related attributes such as:

- Age
- Income
- Employment Details
- Credit Score
- Loan Amount
- Loan Term
- Interest Rate
- Loan Purpose
- Debt Information
- Default Status (Target)
